# Client Brief: VocalStream AI

## 1. Idea del Proyecto
Un agente de voz que procesa audio en tiempo real usando programación asíncrona.

## 2. Arquitectura (Capas)
- **Presentación:** FastAPI (WebSockets) y Next.js.
- **Lógica:** Procesamiento de audio con Asyncio.
- **Datos:** Modelos para transcripciones.

## 3. Validación
Este proyecto valida el uso de streams de baja latencia para aplicaciones de IA.
